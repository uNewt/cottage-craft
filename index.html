INSTALLATION:

  1. Download and Install ATLauncher [18.98MB]
  2. Make sure you've logged into your account. You can verify that you are by ensuring your profile displays on the top right
  3. Press "Instances", then on the left select "Import"
  4. Import the following .zip file, you might need to manually select .mrpack instead of .zip on the bottom right of the file exporer: Remade & Remastered (Custom) 1.21.zip [17.3MB]
